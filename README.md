# ZipInfo PWA

> Zipcode city finder PWA built with Vue.js and Ionic 4 using the @ionic/vue package and the @vue/pwa plugin

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
